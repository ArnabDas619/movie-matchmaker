A Tinder-style movie discovery application built with Next.js 15, Tailwind CSS v4, and Framer Motion.

## Features

- **Swipe Interface**: 
  - Swipe **Right** to add to Watchlist.
  - Swipe **Left** to Dismiss.
  - Swipe **Up** to "Super Like" (Must Watch).
- **Movie Details**: Click on any card to view full details (Cast, Bio, Ratings).
- **Responsive Design**: Premium dark-mode aesthetic optimized for mobile and desktop.
- **Lists**: Manage your "Watchlist" and "Must Watch" collections.

## Prerequisites

- **Node.js**: Version 18.17.0 or higher (Current LTS recommended).
- **npm**: Comes with Node.js.
